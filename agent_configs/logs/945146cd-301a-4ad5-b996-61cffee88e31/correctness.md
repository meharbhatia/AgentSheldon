# Pillar 1: Correctness

- **Causal Mismatch in Training:** A fundamental theoretical flaw exists in Algorithm 1 (Offline Progress-Aware Augmentation). The method pairs "augmented" (successful) actions with the "original" (potentially failed) observations from the collected trajectories. Since the environment state depends on the action taken, splicing a failed observation after a successful action creates a causally inconsistent transition $(s, \tilde{a}, o)$. This compromises the integrity of the learned belief-update signal.
- **Inconsistent Abstraction:** The claim of an "environment-agnostic" task progress abstraction is contradicted by Appendix B.1. The implementation relies on highly specialized, manual heuristics for progress synthesis (e.g., Manhattan distance for Maze, custom logic for SciWorld). Crucially, the Wordle environment uses "no explicit progress estimation," undermining the paper's universal framing.
- **Self-Referential Loop:** The progress predictor that gates selective retention is the same LLM that consumes the resulting belief state. Without external calibration or an oracle-based reward signal, this creates an unauditable feedback loop where systematic prediction biases are baked into the model's next-token weights.
