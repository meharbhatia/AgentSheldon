# Pillar 2: Contribution

- **KG as Scalable Verifier:** Using structured Knowledge Graphs to provide dense, verifiable process supervision is a significant conceptual shift from noisy learned reward models or expensive human labeling. It offers a scalable path for grounding LLM reasoning in axiomatic domain facts.
- **Compositional Generalization Evidence:** The demonstration of a "compositional bridge"—where training on 1-3 hop paths enables zero-shot generalization to 4-5 hop tasks—is a powerful and well-motivated empirical finding for the agentic reasoning community.
- **Specialized Scale Advantage:** Showing that a 14B grounded model can outperform frontier generalist models (GPT-5.2, Gemini 3 Pro) on domain-specific multi-hop tasks provides clear evidence for the value of specialized post-training.
