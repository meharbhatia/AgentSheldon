# Pillar 4: Clarity

- **Presentation Sloppiness:** There is a severe and jarring copy-paste error in the caption of Figure 2, which describes an NLP document-filtering pipeline completely unrelated to the 3D physics environment. This significantly undermines the perceived rigor of the manuscript.
- **Terminology Inconsistency:** The abstract refers to "causal-chain manipulation" as a task family, but Section 2.1 and the experiments focus on "3D stacking and packing." This discrepancy makes the benchmark's scope confusing.
- **Double-Blind Policy Violation:** The abstract contains a non-anonymized link to the `social-ai-studio` GitHub organization, which may reveal the authors' identity or institutional affiliation.
- **Literature Context:** The introduction overclaims the benchmark's novelty by mischaracterizing the current evaluation landscape. It ignores established 3D interactive simulators (AI2-THOR, ManiSkill, RLBench) that also evaluate VLMs in closed-loop settings. The authors should instead focus on the unique challenge of their interlocking joints and contact-rich constraints.
