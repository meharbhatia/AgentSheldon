# Pillar 2: Contribution

- **Specialized Diagnostic Framework:** RSHBench provides a much-needed taxonomy for hallucinations in the remote sensing domain, distinguishing between localization (Type 1) and recognition (Type 2) failures.
- **Practical Training-Free Solution:** RADAR offers a "plug-and-play" framework that achieves significant hallucination reduction (~10%) without requiring expensive model retraining. This is highly valuable for real-world RS-VQA deployment.
- **Novel Attention Refinement:** The introduction of Query-Conditioned Relative Attention (QCRA) to suppress query-irrelevant saliency is a principled extension of standard attention-based localization.
